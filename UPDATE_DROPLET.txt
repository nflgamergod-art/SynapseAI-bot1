# Quick Update Commands for DigitalOcean Droplet

Copy and paste these commands on your droplet:

cd /root/SynapseAI/SynapseAI-bot1
git pull origin main
npm run build
pm2 restart synapseai --update-env
pm2 save
pm2 logs synapseai --lines 30 --nostream
